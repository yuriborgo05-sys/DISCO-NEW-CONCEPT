# 🥂 Bamboo Nightlife PWA | The "Unicorn" Ecosystem

> **Official Digital Infrastructure for TOO MUCH Events at Bamboo Club.**

Benvenuto nel cuore pulsante del **Bamboo Nightlife**. Questa Progressive Web App (PWA) è un ecosistema "Unicorn-Grade" progettato per la gestione chirurgica di serate d'élite, sincronizzando 18 ruoli professionali con un'esperienza VIP senza precedenti.

## 🚀 Key Features

- **⚡ Electronic Warfare Suite**: Controllo folla sincronizzato (Social Hacker Mode & Global Strobe).
- **🔮 AI Oracle v3**: Assistente virtuale con knowledge base master e privacy gates per la direzione.
- **🗺️ 18-Role Routing**: Matrice di coordinamento totale per PR, Security, Cassa, Immagine, Fotografi e altro.
- **🍾 Smart Bottle Service**: Ordini 1:1 in tempo reale con validazione QR univoca.
- **✨ AR Hostess Experience**: Selezione hostess in Realtà Aumentata (WebXR).
- **🛡️ Military-Grade Security**: Firestore Hardened Security Rules & Persistence Offline.

## 🛠️ Tecnologie

- **Frontend**: React 18 + Vite (Sartorial UI Design)
- **Backend**: Firebase Firestore & Authentication
- **AR/XR**: Model Viewer / WebXR
- **UI/UX**: Glassmorphism / Neon Accents / Lucide Icons

## ⛓️ Setup Rapido

1.  **Configura Firebase**: Inserisci le tue API Key in `src/firebase.js`.
2.  **Sicurezza**: Carica le regole contenute in `DATABASE_HARDENING.md` nella tua console Firebase.
3.  **Lancio**: Esegui `npm install` e poi `npm run dev`.
